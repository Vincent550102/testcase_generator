# testcase_generator

## 簡介
此工具幫助製作程式題目的測試資料。你需要準備以下事項
1. 將標準程式寫好，並編譯成可執行檔，在執行工具時將路徑放入參數。
2. 將測試資料以附檔名為 .in 的純文字檔放入 in 資料夾內。
3. 按照 #用法 執行程式

執行完工具你會得到
1. 根據 in 資料夾內的標準輸入對應的標準輸出


## 用法
```
python testcase_generator.py <excutable file>
```
ex.
```
python testcase_generator.py solve.exe
```
